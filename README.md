Real Estate Appointment Automation POC
Overview
An intelligent real estate platform automating appointment scheduling via Telegram Bot and AI.

Features
AI-Powered Scheduling: Uses OpenAI GPT-4o for natural language understanding.
Real-Time Availability: Syncs with Google Calendar automatically.
RAG Architecture: Vector-based search for property listings.
Tech Stack
Frontend: React.js
Backend/Orchestration: n8n (Self-hosted)
Database: PostgreSQL + pgvector
Infrastructure: Docker & Docker Compose
